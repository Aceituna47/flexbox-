Este es un ejemplo del uso de CSS Shape Outside. Normalmente el texto rodea una imagen cuadrada, pero con esta propiedad podemos lograr que se ajuste a un círculo u otras formas personalizadas. Es útil para hacer maquetados más creativos en páginas web, dándole un estilo más dinámico y menos rígido que los rectángulos habituales. Se puede usar con círculos, polígonos o incluso formas personalizadas.
